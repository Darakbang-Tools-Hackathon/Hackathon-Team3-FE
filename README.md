# PoseUp: 스마트 기상과 AI 챌린지 앱

> 프로젝트 비전: REM 수면 사이클 기반의 스마트 알람과 AI 포즈 인증 챌린지를 결합하여, 건강한 아침 습관 형성을 지원합니다.
> 

## 1. 핵심 기능 모듈 (Core Functional Modules)

### 1.1. ⏰ 스마트 기상 시스템 (Smart Wake-up)

| 기능 | 상세 요약 |
| --- | --- |
| **REM 주기 추천** | 잠들 시간 입력 시, 90분 주기를 계산하여 **최적의 기상 시간 3가지**를 제안합니다 (잠들기까지 15분 고려). |
| **FCM 알림 발송** | 설정된 기상 시간에 맞춰 사용자에게 **챌린지 알림을 푸시 발송**하는 백엔드 스케줄링 시스템이 구현되었습니다. |

### 1.2. 🧘 AI 포즈 인증 챌린지 (AI Pose Challenge)

| 기능 | 상세 요약 |
| --- | --- |
| **챌린지 결과 처리** | 성공/실패 결과를 받아 **LP, 랭킹, 이력 데이터**를 **트랜잭션으로 안전하게 업데이트**하는 백엔드 코어 로직이 구현되었습니다. |
| **실시간 피드백** | 포즈 정확도에 따라 빨강/노랑/녹색으로 구분된 **상태 메시지 및 진행도**를 제공하는 UI/UX가 구축되었습니다. |

### 1.3. 🏆 게임화 및 업적 시스템 (Gamification & Achievements)

| 기능 | 상세 요약 |
| --- | --- |
| **LP 및 레벨 시스템** | 성공 시 +100 LP, 실패 시 -10 LP가 반영되며, 500 LP당 레벨업 (새싹 → 해바라기 → 태양)이 진행됩니다. |
| **6가지 팀 업적 시스템** | **숙면의 증명** (누적 성공 100회), **팀워크 전문가** (4주 연속 목표 달성) 등 복잡한 검증 로직이 **백엔드에서 자동으로 처리 및 보상 지급**됩니다. |

### 1.4. 🤝 소셜 및 팀 관리 (Team & Social Management)

| 기능 | 상세 요약 |
| --- | --- |
| **팀 코어 관리** | 팀 생성/참여/탈퇴, 팀장 위임, 팀원 강퇴 등 **모든 팀장 권한 로직**이 백엔드에서 구현되었습니다. |
| **주간 랭킹 시스템** | 매주 월요일 KST 자정에 주간 랭킹을 산정하고 **경쟁 업적을 자동 지급**하는 배치(Batch) 시스템이 구축되었습니다. |
| **인증 및 DB 관리** | 회원가입/탈퇴 시 유저 문서 생성/삭제가 **Firebase Authentication 트리거**를 통해 자동으로 실행됩니다. |

## 2. 기술 스택 및 상태 (Tech Stack & Status)

| 영역 | 기술 스택 |
| --- | --- |
| **Frontend Platform** | **Flutter (Dart)** |
| **Backend / DB** | **JavaScript + Firebase** (Functions, Firestore, Auth 기반) |
| **ML Integration** | ML Kit (연동을 위한 UI/로직 기반 구축 완료) |
