# ☀️ PoseUp : 잠들고, 일어나서 포즈!

> 핵심 비전: REM 수면 사이클 기반 스마트 알람과 AI 포즈 인증 챌린지를 결합하여, '개운한 아침'을 만드는 건강 습관 형성 소셜 챌린지 앱.
> 

현재 **React 프로토타입** 개발이 완료되었으며, 모든 UI/UX 플로우와 게임화 시스템 로직이 구현되어 ML Kit 및 백엔드 연동을 위한 준비를 마쳤습니다.

## ⭐ 1. PoseUp 핵심 구현 기능 (The Core)

프로토타입은 세 가지 핵심 시스템을 중심으로 구축되었습니다.

### ⏰ 스마트 알람 시스템 (Onboarding Screen)

- **REM 수면주기 기반 추천**: 사용자가 잠들 시간을 선택하면, 90분 수면 주기를 계산하여 최적의 기상 시간 3가지(4주기, 5주기 권장, 6주기)를 제안합니다.
- **정교한 계산**: 잠들기까지 필요한 15분을 고려하여 정확한 기상 시간을 추천합니다.

### 🧘 AI 포즈 챌린지 시스템 (Challenge Screen)

- **ML Kit Ready UI**: 카메라 프리뷰, 5분 카운트다운 타이머, 스켈레톤 포즈 가이드 오버레이 등 실제 ML Kit 연동을 위한 모든 UI를 구현했습니다.
- **실시간 피드백**: 챌린지 진행도(0-100%)에 따라 실시간으로 피드백을 제공합니다.
    - **0-30%**: "포즈를 확인해주세요" (빨강)
    - **30-70%**: "좋아요! 조금만 더 정확하게!" (노랑)
    - **70-100%**: "완벽해요! 자세를 유지하세요!" (녹색)
- **결과 보상**: 성공 시 Confetti 애니메이션 및 LP 변동(카운트업)을 시각적으로 제공합니다.

### 🏆 게임화 및 소셜 시스템 (Gamification & Team)

- **LP (League Points)**: 성공 시 **+100 LP**, 실패 시 **10 LP**를 부여하며, 500 LP당 레벨업합니다.
- **캐릭터 진화**: 레벨에 따라 🌱 새싹 → 🌻 해바라기 → 🌞 태양 순으로 캐릭터가 시각적으로 진화합니다.
- **팀 경쟁**: 주간 팀 랭킹, 팀 목표 진행도, 팀원별 성공/실패 상태를 실시간으로 확인하며 경쟁 및 격려합니다.

## 📱 2. 구현된 화면 플로우 (8 Screens)

| 플로우 | 화면 | 주요 역할 |
| --- | --- | --- |
| **시작** | Splash, Login | 로딩 및 소셜 로그인 연동 UI, 약관 동의 |
| **설정** | Onboarding | 수면 시간 입력 및 REM 기반 최적 기상 시간 추천 |
| **메인** | Main Dashboard | 사용자 레벨/LP, 통계(스트릭, 성공률), 주간 그래프 요약 |
| **실행** | Challenge Screen | AI 포즈 인증 챌린지 실행 및 실시간 피드백 |
| **결과** | Result Screen | 성공/실패 결과, LP 변동 애니메이션, 맞춤 팁 제공 |
| **소셜** | Team Create/Join | 팀 생성(자동 코드) 및 팀 코드 입력으로 참여 |
| **소셜** | Team Dashboard | 팀 순위, 총 LP, 팀원 현황 및 주간 목표 진행도 |

## ⚙️ 3. 기술 스택 및 디자인 시스템

### 3.1. 기술 스택 (Tech Stack)

- **Frontend**: `React` + `TypeScript`
- **Styling**: `Tailwind CSS` (v4.0) 및 `Shadcn UI` 컴포넌트
- **Iconography**: `Lucide React`
- **배포 환경**: PhoneFrame 기반 모바일 UI 최적화 완료

### 3.2. 디자인 시스템

- **메인 컬러**: `#FFA858` (따뜻한 오렌지)
- **상태 컬러**: 성공 (`#81C784`), 실패 (`#EF5350`)
- **성장 시각화**: 레벨 구간별 (Lv.1-4, Lv.5-9, Lv.10+) 그라데이션 변화를 적용하여 성장을 즉각적으로 보상합니다.

## 🚧 4. 향후 로드맵 (Next Steps)

프로토타입 위에 실제 배포를 위한 네이티브 및 백엔드 기능을 통합할 예정입니다.

4. **FCM 푸시 알림 구현**: 알람 시간 푸시 발송 및 팀원 활동 알림 기능 통합.
