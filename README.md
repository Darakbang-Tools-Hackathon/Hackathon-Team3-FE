# PoseWake 프로젝트 구현 기능 보고서 (Prototype V1.0)

## 1. 프로젝트 개요

### 1.1. 핵심 컨셉

PoseWake는 **REM 수면 사이클 기반의 스마트 알람** 기능과 **AI 포즈 인증 챌린지**를 결합하여 사용자에게 '잘 자고, 개운하게 일어나도록' 돕는 건강 습관 형성 소셜 챌린지 모바일 애플리케이션입니다. 사용자 동기 부여를 극대화하기 위해 게임화(Gamification) 시스템을 핵심 요소로 도입하였습니다.

### 1.2. 목표 및 구현 상태

React 기반의 프로토타입 개발을 통해 핵심 사용자 플로우(User Flow) 및 모든 UI/UX 디자인 시스템 구축을 완료했습니다. 현재 ML Kit 및 백엔드 연동을 위한 모든 프론트엔드 기반 작업이 완료된 상태입니다.

## 2. 구현된 핵심 시스템 및 기능

프로토타입은 총 8개의 스크린을 통해 사용자 등록, 온보딩, 챌린지 실행 및 소셜 활동에 이르는 완벽한 엔드투엔드(End-to-End) 플로우를 지원합니다.

### 2.1. 사용자 플로우 & 온보딩 (Phase I)

| 화면 | 주요 기능 | 비고 |
| --- | --- | --- |
| **Splash Screen** | 앱 로딩 및 브랜딩 애니메이션 | 2초 후 자동 전환 로직 구현 |
| **Login Screen** | 소셜 로그인 기반의 초기 접근 | Google, Apple 소셜 로그인 버튼 및 이용약관 동의 UI |
| **Onboarding Screen (⭐ 핵심)** | **REM 수면 주기 추천 시스템** | 사용자가 잠들 시간 입력 시, 90분 주기 기반 4/5/6 주기 최적 기상 시간 3가지 추천 (잠들기까지 15분 고려) |

### 2.2. 메인 대시보드 및 통계 (Phase II)

| 구성 요소 | 기능 상세 | 목적 |
| --- | --- | --- |
| **사용자 정보** | 이름, 레벨 (Lv. 1, 5, 10), LP (League Points), 레벨 진행 바 | 시각적 성장 및 동기 부여 |
| **통계 카드** | 연속 성공 일수 (스트릭), 주간 성공률 (%), 달성 스트릭 (총 일수) 표시 | 성과 요약 및 행동 유도 |
| **기록 그래프** | 주간 (요일별) 성공/실패 막대 그래프 | 시각적 성과 분석 제공 |
| **다음 챌린지** | 설정된 알람 시간 표시 및 챌린지 시작 버튼 | 메인 콜투액션 (CTA) |

### 2.3. AI 포즈 챌린지 시스템 (Phase III: ML Kit Ready)

챌린지 화면은 실제 ML Kit 연동에 필요한 모든 UI/UX 요소를 완벽히 구현하여, 실제 SDK 통합 시 즉시 배포 가능하도록 설계되었습니다.

| 요소 | 상세 구현 내용 |
| --- | --- |
| **카메라 시뮬레이션** | 실제 카메라 프리뷰 영역 UI, 녹화 중 (빨간 점) 애니메이션 |
| **챌린지 타이머** | 5분 카운트다운 타이머 및 진행 바로 시각화 |
| **포즈 가이드라인** | 스켈레톤 형태의 포즈 가이드 오버레이 (숨기기/다시보기 기능 포함) |
| **실시간 피드백** | 챌린지 진행도 바 (0-100%) 및 상태별 메시지 제공 |
| **상태별 피드백 로직** | **0-30%**: "포즈를 확인해주세요" (빨강) / **30-70%**: "좋아요! 조금만 더 정확하게!" (노랑) / **70-100%**: "완벽해요! 자세를 유지하세요!" (녹색) |

### 2.4. 결과 및 피드백 (Result Screen)

- **결과 표시**: 성공 (녹색 체크, Confetti 애니메이션), 실패 (빨간색 X, 격려 메시지)
- **LP 변동**: 성공 시 **+100 LP** (카운트업 애니메이션), 실패 시 **10 LP**
- **추가 통계**: 완벽도 (95%), 순위 (#12) 등 상세 결과 보고
- **맞춤 팁**: 성공/실패 상황에 맞는 건강/수면 조언 제공

### 2.5. 소셜 및 팀 시스템 (Team Feature)

| 화면 | 주요 기능 및 특징 |
| --- | --- |
| **Team Dashboard** | 팀 이름, 코드, 팀 순위 (#3), 총 LP, 주간 목표 진행도 (70%) 표시 |
| **팀원 현황** | 각 팀원의 레벨, LP, 연속 성공 일수, 오늘의 챌린지 상태 (✓, ✗, ⏳) 실시간 표시 |
| **팀 경쟁 현황** | 주간 팀 랭킹 (Top 4 팀) 및 2위까지 필요한 LP 차이 표시 |
| **Team Create/Join** | 새 팀 만들기 (자동 팀 코드 생성: WAKE + 랜덤 6자리) 및 코드 입력으로 참여 기능 |

## 3. 게임화 및 동기 부여 시스템 상세 (Gamification System)

PoseWake의 핵심은 지속 가능한 습관 형성을 위한 강력한 게임 요소입니다.

| 시스템 | 상세 로직 | 보상/징벌 |
| --- | --- | --- |
| **LP (League Points)** | 챌린지 성공 및 업적 달성 시 획득 | 성공: +100 LP / 실패: -10 LP |
| **레벨 시스템** | 500 LP당 레벨업 | **Lv.1-4**: 🌱 새싹 / **Lv.5-9**: 🌻 해바라기 / **Lv.10+**: 🌞 태양 |
| **업적** | 5일 연속 성공, 주간 성공률 85% 달성, 달성 스트릭 등 | 추가 LP 지급 및 시각적 배지 |
| **팀 경쟁** | 주간 랭킹 경쟁 및 팀 목표 (LP) 달성 | 보너스 LP 및 팀 순위 뱃지 |

## 5. 향후 로드맵 (Roadmap)

현재 프로토타입을 기반으로 최종 배포를 위해 다음 단계의 개발이 필요합니다.

1. **ML Kit Pose Detection 실제 연동 및 알고리즘 구현**
    - 네이티브 카메라 API 접근 및 실시간 비디오 스트림 처리.
    - 관절 좌표 데이터를 활용한 포즈 정확도 계산 및 피드백 로직 통합.
2. **백엔드(Backend) 시스템 연동**
    - 사용자 인증 및 데이터 (LP, 레벨, 통계) 동기화.
    - 팀 시스템 및 실시간 리더보드 데이터 처리.
3. **FCM 푸시 알림 서비스 구축**
    - 알람 시간에 맞춘 챌린지 푸시 알림 발송 기능.
    - 팀원 활동에 대한 실시간 알림 기능.
4. **Flutter/Dart 플랫폼 전환**
    - React 프로토타입을 Flutter로 포팅하여 네이티브 퍼포먼스 및 카메라/ML Kit 연동 최적화.
    - 글로벌 서비스를 위한 다국어 지원 (i18n) 통합.
